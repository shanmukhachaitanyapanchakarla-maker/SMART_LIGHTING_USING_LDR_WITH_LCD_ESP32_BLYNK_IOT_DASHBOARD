# SMART_LIGHTING_USING_LDR_WITH_LCD_ESP32_BLYNK_IOT_DASHBOARD
Smart lighting using LDR with LCD, ESP32, and Blynk IOT Dashboard is a real time application useful for monitoring light intensity and control the light based on light intensity with IOT Dashboard.
As part of project task for Maincrafts embedded systems and IOT internship. 

## 📖 Project Overview

The Smart Lighting System Using LDR with LCD, ESP32, and Blynk IOT Dashboard is an advancement in previous embedded systems project. it is designed with ESP32 microcontroller for adding WIFI to the system. The system utilizes a Light Dependent Resistor (LDR) to sense environmental light intensity and intelligently switch a lighting device ON or OFF and prints the state of the system on LCD display and uptades the state of the system in the IOT dashboard. This automation helps reduce energy consumption and improves lighting efficiency and user friendly.

## ⚙️ Working Principle

The system continuously monitors ambient light intensity using an LDR sensor. The sensor's resistance changes according to the amount of light falling on it.

- During daylight conditions, the system keeps the light source OFF.
- During low-light or dark conditions, the system automatically turns the light source ON.
- Display the state of the light bulb on LCD display, whether it is ON or OFF.
- Update the IOT Dashboard and reports the status of the system.
